# cli-opentui

OpenTUI (SolidJS for CLI) design and implementation toolkit.

## Overview

This skill provides patterns, best practices, and implementation guidance for building CLI applications with OpenTUI and SolidJS.

## Contents

- **agents/**: AI agent configurations for component architecture
- **commands/**: Skill invocation commands
- **skills/opentui-design/**: Core design patterns and references
  - **references/**: Detailed documentation on specific topics
  - **examples/**: Code examples and templates

## Key Topics

- Component patterns for OpenTUI
- Input handling and key propagation prevention
- Mouse handling and limitations
- Screen navigation patterns
- State management with SolidJS
- Performance optimization
- Common gotchas and workarounds

## Usage

This skill is designed to be used with Claude Code when building CLI applications using OpenTUI/SolidJS stack.
