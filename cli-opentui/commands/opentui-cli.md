# /opentui-cli

Invoke OpenTUI CLI design patterns and component guidance.

## Usage

```
/opentui-cli [topic]
```

## Topics

- `component` - Component design patterns
- `input` - Input handling and key propagation
- `mouse` - Mouse handling and limitations
- `navigation` - Screen navigation patterns
- `state` - State management with SolidJS
- `gotchas` - Common issues and workarounds

## Examples

```
/opentui-cli input
/opentui-cli gotchas
/opentui-cli component
```
